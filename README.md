# lightning
> 一个基于vue2.0的电影web app项目

后端：Node.js
服务器：express 4.16.4
数据库：MongoDB 4.0.3
mongoose
前端：vue 2.9.6
第三方接口：时光网资讯API，豆瓣电影API
+ axios
+ element-ui
+ vue-awesome-swiper
+ vue-calendar-component

#### 已完成功能
- :heavy_check_mark: 用户登录&注册
- :heavy_check_mark: 修改密码
- :heavy_check_mark: 电影资讯首页轮播
- :heavy_check_mark: 电影资讯详情页
- :heavy_check_mark: 院线热映
